## Мой реферат по предмету "Математическое моделирование"

Часть реферата по предмету "Математическое моделирование".

Данный репозиторий содержит (будет содержать) визуализацию задачи поиска оптимального (кратчайшего) маршрута во взвешенном неориентированном графе.


## Детали реализации
### Plan:
- framework для визуализации: [vis.js](https://visjs.org/). Подходит для графов
- хранение графа в виде **списков смежности**. Подъодит потому что граф разреженный
- алгоритм: Дейкстра. Планирую ещё попробовать апгрейдить её с кучей
