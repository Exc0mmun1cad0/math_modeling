## Мой реферат по предмету "Математическое моделирование"

Часть реферата по предмету "Математическое моделирование".

Данный репозиторий содержит визуализацию задачи поиска оптимального (кратчайшего) маршрута во взвешенном неориентированном графе.


## Детали реализации
### Plan:
- библиотека для визуализации: [vis.js](https://visjs.org/). Подходит для графов
- хранение графа в виде **списков смежности**. Подходит потому что граф разреженный
- алгоритм: Дейкстра c кучей и без
